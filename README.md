# TL;DR Block

This block finds url records in your base, and summarizes them using SMMRY.com.
The TL;DR is then saved back in your base.

Constraints:
* A Table named Urls
* A column named URL -> URL to summarize
* A column named Summary -> TL;DR Goes here
